# FilmFlip icon sources

The interface icons in this folder are downloaded as SVG from the official Iconify API.

- UI and metadata icons: Phosphor Duotone by Phosphor Icons, MIT License.
  - https://icon-sets.iconify.design/ph/
  - https://github.com/phosphor-icons/core/blob/main/LICENSE
- FilmFlip logo: `boxicons:film-roll-filled` from Boxicons,
  customized with the FilmFlip palette and film perforations. MIT License.
  - https://icon-sets.iconify.design/boxicons/film-roll-filled/
  - https://github.com/box-icons/boxicons-core/blob/main/LICENSE

- Message dialog icons: `ph:info-duotone`, `ph:warning-circle-duotone`,
  `ph:x-circle-duotone`, and `ph:question-duotone` from Phosphor Icons.
  - https://icon-sets.iconify.design/ph/

The SVGs are stored locally so the application does not need a network connection at runtime.
