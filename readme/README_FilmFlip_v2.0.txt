===============================================================================
FilmFlip v2.0 사용자 설명서
필름 스캔 이미지 정리 · 이름 변경 · EXIF 편집 도구
===============================================================================


1. FilmFlip 소개
-------------------------------------------------------------------------------

FilmFlip은 한 롤의 필름 스캔 이미지를 한 번에 확인하고 정리하는 프로그램입니다.

주요 기능

- 폴더 안의 JPG, JPEG, PNG 이미지를 썸네일로 확인
- 이미지 파일명을 현재 순서 또는 역순으로 일괄 변경
- 촬영 정보를 조합해 파일명과 폴더명 생성
- 촬영 프리셋과 카메라·필름·현상소·장소·스캐너 목록 관리
- JPG/JPEG 파일의 실제 EXIF 정보 변경
- 마지막 파일명 또는 폴더명 변경 작업 Undo
- 일반 모드와 다크 모드
- 24·27·36·72컷 및 사용자 지정 롤 기준 컷수


2. 설치와 실행
-------------------------------------------------------------------------------

macOS

1) FilmFlip_V2.0_mac.zip의 압축을 풉니다.
2) FilmFlip.app을 응용 프로그램 폴더로 옮기는 것을 권장합니다.
3) FilmFlip.app을 더블 클릭해 실행합니다.

macOS가 개발자를 확인할수 없다는 안내를 표시하는 경우 (휴지통으로 이동과 완료가 뜰겁니다)

1) 완료를 누른뒤 시스템 설정을 엽니다.
2) 개인정보 보호 및 보안으로 이동합니다.
3) FilmFlip에 대해 '그래도 열기'를 선택합니다.

Windows

1) FilmFlip_V2.0_win.zip의 압축을 풉니다.
2) FilmFlip.exe를 더블 클릭해 실행합니다.

Windows가 PC 보호 안내를 표시하는 경우

1) '추가 정보'를 누릅니다.
2) 파일 이름이 FilmFlip.exe인지 확인합니다.
3) '실행'을 선택합니다.

압축 파일 안에서 프로그램을 직접 실행하지 말고 먼저 압축을 풀어 주세요.


3. 빠른 시작
-------------------------------------------------------------------------------

1) 상단의 '폴더 선택'을 누릅니다.
2) 스캔 이미지가 들어 있는 폴더를 선택합니다.
3) 가운데 목록에서 썸네일, 현재 파일명, 변경될 파일명을 확인합니다.
4) 필요하면 상단의 '역순 변경'으로 바로 역순 이름 변경을 실행합니다.
5) 세부 설정이 필요하면 '이름 변경'을 누릅니다.
6) 실행 전에 미리보기를 확인합니다.

중요한 사진은 작업 전에 별도의 폴더에 복사해 두는 것을 권장합니다.


4. 메인 화면
-------------------------------------------------------------------------------

폴더 선택

- 이미지가 들어 있는 폴더를 불러옵니다.
- 지원 이미지가 없으면 빈 목록으로 표시됩니다.

역순 변경

- 현재 설정된 번호 자릿수로 이미지 번호를 역순 변경합니다.
- 예: 36장인 경우 첫 이미지가 036, 마지막 이미지가 001이 됩니다.

이름 변경

- 접두 텍스트, 번호, 정렬 방식 등을 지정하는 이름 변경 창을 엽니다.
- 촬영 정보를 조합하는 상세 이름 변경 화면도 사용할 수 있습니다.

Undo

- FilmFlip에서 마지막으로 실행한 파일명 또는 폴더명 변경을 되돌립니다.
- 앱을 종료하거나 다른 작업을 실행하면 이전 작업을 되돌릴 수 없을 수 있습니다.
- EXIF 변경은 Undo 대상이 아닙니다.

일반 모드 / 다크 모드

- 오른쪽 위 해 또는 달 아이콘으로 화면 테마를 변경합니다.

설정

- 화면 테마, 썸네일 크기, 롤 기준 컷수, 기본 정렬 방식을 설정합니다.


5. 가운데 이미지 목록과 미리보기
-------------------------------------------------------------------------------

가운데 목록에는 다음 정보가 표시됩니다.

- 썸네일
- 현재 파일명
- 변경될 파일명
- 상태

이미지를 선택하면 오른쪽 미리보기에서 사진, 파일명, 해상도, 형식을 확인할 수
있습니다.

썸네일을 더 크게 보고 싶으면 왼쪽의 '썸네일 크기'에서 작게·보통·크게 중 하나를
선택합니다.

오른쪽 롤 요약에는 다음 정보가 표시됩니다.

- 파일 수
- 현재 정렬 방식
- 변경될 첫 파일명
- 변경될 마지막 파일명


6. 롤 정보
-------------------------------------------------------------------------------

왼쪽 롤 정보에는 다음 항목을 표시할 수 있습니다.

- 카메라
- 렌즈
- 필름
- ISO
- 촬영일
- 촬영장소
- 현상소
- 스캐너
- 메모

롤 정보는 촬영 프리셋과 이름 변경·폴더명 변경에 사용하는 정보입니다.

카메라, 렌즈, 필름, 현상소, 스캐너 같은 값이 왼쪽에 표시된다고 해서 사진의
EXIF에 자동으로 기록되는 것은 아닙니다. 실제 EXIF를 바꾸려면 'EXIF 정보 변경'을
사용해야 합니다.


7. 파일명 변경
-------------------------------------------------------------------------------

메인 화면 오른쪽 아래의 '이름 변경 실행' 또는 상단의 '이름 변경'을 누릅니다.

간단 이름 변경 설정

- 접두 텍스트: 번호 앞에 붙일 글자
- 숫자 자릿수: 1, 01, 001처럼 번호 길이 지정
- 시작 번호: 첫 번호 지정
- 정렬 방식: 현재 순서 또는 역순
- 확장자: 원래 확장자 유지

예시

- 접두 텍스트: A_
- 숫자 자릿수: 3
- 시작 번호: 1
- 결과: A_001.jpg, A_002.jpg, A_003.jpg ...

촬영 정보를 포함한 이름 변경

1) 촬영 프리셋을 선택하거나 카메라·필름·현상소·장소를 직접 입력합니다.
2) 필요하면 '추가 정보'를 열어 날짜·스캐너·메모를 입력합니다.
3) 파일명 구성에서 사용할 항목을 체크합니다.
4) ▲ 위로 / ▼ 아래로 버튼으로 항목 순서를 정합니다.
5) 현재 순서 또는 역순을 선택합니다.
6) 미리보기를 확인합니다.
7) '이름 변경'을 누릅니다.

파일명에 사용할 수 없는 문자는 자동으로 정리될 수 있습니다.


8. 촬영 프리셋과 항목 관리
-------------------------------------------------------------------------------

자주 사용하는 촬영 조합을 촬영 프리셋으로 저장할 수 있습니다.

예시

- Nikon FM2 + Kodak Vision3 250D + 망우삼림
- EOS 5 + ColorPlus 200 + 다크룸

촬영 프리셋 버튼

- 저장: 현재 입력한 촬영 정보를 새 프리셋으로 저장
- 수정: 선택한 프리셋을 현재 값으로 수정
- 삭제: 선택한 프리셋 삭제
- 편집: 프리셋 이름과 순서 관리

각 항목 옆의 '관리'를 누르면 카메라, 필름, 현상소, 장소, 스캐너 목록을 저장·수정·
삭제하고 순서를 바꿀 수 있습니다.


9. 폴더명 변경
-------------------------------------------------------------------------------

메인 화면 가운데 아래의 '폴더명 변경'을 누릅니다.

1) 촬영 프리셋을 선택하거나 날짜·메모·카메라·필름·현상소·장소·스캐너를
   입력합니다.
2) 폴더명 구성에서 사용할 항목을 체크합니다.
3) ▲ 위로 / ▼ 아래로 버튼으로 항목 순서를 정합니다.
4) 변경될 폴더명 미리보기를 확인합니다.
5) '폴더명 변경'을 누릅니다.

폴더명이 바뀐 뒤에도 FilmFlip은 변경된 폴더를 계속 사용합니다.


10. EXIF 정보 변경
-------------------------------------------------------------------------------

왼쪽의 'EXIF 정보 변경'을 누르면 현재 폴더의 JPG/JPEG 사진에 실제 EXIF 정보를
기록할 수 있습니다.

지원 항목

- 촬영일
- 카메라 제조사
- 카메라 모델
- 렌즈 모델
- ISO
- 조리개
- 셔터 속도
- 초점 거리
- 촬영자
- 저작권
- 이미지 설명
- 사용자 메모

사용 방법

1) EXIF 정보 변경을 누릅니다.
2) 적용 대상의 JPEG 수를 확인합니다.
3) 필요한 항목을 입력합니다.
4) '빈 칸은 기존 EXIF 값을 유지' 여부를 확인합니다.
5) 'EXIF 정보 적용'을 누릅니다.
6) 완료 안내가 나올 때까지 기다립니다.

날짜만 지정하면 사진의 EXIF에는 해당 날짜와 00:00:00 시간이 기록됩니다.

'빈 칸은 기존 EXIF 값을 유지'가 체크되어 있으면 입력하지 않은 항목은 바뀌지
않습니다. 체크를 해제하면 빈 항목의 기존 값이 제거될 수 있으므로 주의하세요.

PNG 파일은 EXIF 변경 대상에서 자동으로 제외됩니다.


11. 설정
-------------------------------------------------------------------------------

오른쪽 위 설정 아이콘을 누릅니다.

테마

- 일반 모드
- 다크 모드

썸네일 크기

- 작게
- 보통
- 크게

롤 기준 컷수

- 24컷
- 27컷
- 36컷
- 72컷(하프)
- 원하는 숫자 직접 입력

실제 이미지 수가 기준 컷수보다 많으면 메인 ROLL 카드에 초과 컷수가 함께
표시됩니다. 롤 기준 컷수를 바꿔도 사진 파일이나 썸네일 자체는 변경되지 않습니다.

기본 정렬

- 역순
- 현재 순서 유지

마지막으로 선택한 폴더 기억

- 체크하면 다음 실행 때 마지막 선택 폴더 정보를 기억합니다.


12. 지원 파일과 주의사항
-------------------------------------------------------------------------------

목록과 이름 변경 지원

- JPG
- JPEG
- PNG

EXIF 변경 지원

- JPG
- JPEG

주의사항

- FilmFlip v2.0은 이름 변경이나 EXIF 변경 전에 .bak 백업 파일을 만들지 않습니다.
- EXIF 변경은 Undo로 되돌릴 수 없습니다.
- 같은 폴더에 변경될 이름과 동일한 파일이 있으면 작업이 실패할 수 있습니다.
- 읽기 전용 폴더나 권한이 없는 폴더에서는 변경 작업이 실패할 수 있습니다.
- 클라우드 동기화 중인 폴더는 동기화가 끝난 뒤 작업하는 것을 권장합니다.
- 중요한 원본은 별도 저장장치나 다른 폴더에 복사해 두는 것이 안전합니다.


13. 문제가 생겼을 때
-------------------------------------------------------------------------------

이미지가 표시되지 않을 때

- 지원 형식인지 확인합니다.
- 압축 파일 안의 이미지가 아니라 실제 폴더를 선택했는지 확인합니다.
- 폴더 읽기 권한이 있는지 확인합니다.

썸네일에 '로딩 중'이 오래 표시될 때

- 이미지 수와 파일 크기에 따라 잠시 시간이 필요할 수 있습니다.
- 폴더를 다시 선택해 보세요.

이름 변경 버튼이 비활성화될 때

- 먼저 이미지 폴더를 선택해야 합니다.
- 지원 이미지가 한 장 이상 있어야 합니다.

EXIF 적용이 되지 않을 때

- JPG/JPEG 파일인지 확인합니다.
- 파일이 읽기 전용인지 확인합니다.
- 다른 프로그램에서 해당 파일을 사용 중인지 확인합니다.
