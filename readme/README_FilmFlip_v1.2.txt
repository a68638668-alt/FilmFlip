🎞 FilmFlip v1.2

프로그램 소개

FilmFlip은 필름 스캔 이미지를 원하는 순서로 정렬하고, 촬영 메타데이터를
이용해 파일명과 폴더명을 한 번에 정리할 수 있는 필름 스캔 파일 관리
도구입니다.

v1.2에서는 Roll Metadata, 폴더명 변경 기능, Undo 개선, 사용성 및 성능
향상에 중점을 두었습니다.

------------------------------------------------------------------------

주요 기능

-   폴더 선택 및 드래그 앤 드롭
-   썸네일 미리보기
-   사진 확대 보기
-   사진 순서 변경
-   파일명 변경
-   폴더명 변경
-   Roll Metadata 지원 (카메라, 필름, 현상소, 장소, 날짜, 스캐너, 메모, 파일명번호)
-   촬영 프리셋
-   파일명 / 폴더명 Undo
-   Finder에서 현재 폴더 바로 열기

------------------------------------------------------------------------

설치 및 실행

macOS

1.  압축을 해제합니다.
2.  FilmFlip.app을 실행합니다.
3.  처음 실행 시 보안 경고가 나타나면 휴지통으로 이동이 아닌 '완료' 누른뒤 시스템 설정 → 개인정보 보호 및
    보안 하단으로 이동 → 그래도 열기를 선택합니다. (*처음한번만 하면됩니다)

Windows

FilmFlip.exe 실행

SmartScreen 경고가 나타나면

추가 정보 → 실행

을 선택합니다.

------------------------------------------------------------------------

사용 방법

1. 폴더 불러오기

-   폴더 선택
-   드래그 앤 드롭

2. 사진 순서 변경

-   행 이동
-   역순 변경

3. 파일명 변경

지원 메타데이터

-   날짜
-   카메라
-   렌즈
-   필름
-   ISO
-   현상소
-   장소
-   스캐너
-   메모
-   번호

4. 폴더명 변경

원하는 메타데이터를 선택하여 폴더명을 생성할 수 있습니다.

예)

2026-07-01_남이섬

또는

2026-07-01_홍길동

5. 촬영 프리셋

저장

불러오기

수정

삭제

순서 변경

6. Undo

마지막 작업 1회를 되돌립니다.

-   파일명 변경
-   폴더명 변경
-   파일명 + 폴더명 변경

------------------------------------------------------------------------

주의 사항

-   작업 전 백업 권장
-   Undo는 마지막 작업만 지원
-   외부에서 파일/폴더를 이동하면 Undo가 실패할 수 있습니다.
