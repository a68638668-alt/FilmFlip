# 🎞 FilmFlip v1.1

FilmFlip은 필름 스캔 이미지를 원하는 순서로 정렬하고, 촬영 정보와 번호를 조합해 파일명을 한 번에 변경할 수 있는 필름스캔 파일정리 도구입니다.

v1.1에서는 기존 역순 정렬 기능을 유지하면서, 파일명 구성 UI, 촬영 정보 관리, 촬영 프리셋 관리, 썸네일 표시 안정성, 한글 입력 처리, 실행 성능을 개선했습니다.

------------------------------------------------------------------------

## 주요 기능

- 폴더 선택 또는 드래그 앤 드롭으로 이미지 불러오기
- 썸네일 미리보기
- 썸네일 더블클릭 확대 보기
- 행 드래그로 사진 순서 직접 변경
- 역순 변경 버튼으로 전체 순서 뒤집기
- 카메라 / 필름 / 현상소 / 장소 정보를 활용한 파일명 생성
- 파일명 구성 항목 선택 및 순서 변경
- 카메라 / 필름 / 현상소 / 장소 목록 저장 및 관리
- 촬영 프리셋 저장, 불러오기, 수정, 삭제, 순서 변경
- 마지막 이름 변경 작업 Undo
- macOS 점파일 및 ._ 리소스 파일 무시

------------------------------------------------------------------------

## 사용 방법

### 1. 프로그램 실행

macOS에서는 `FilmFlip.app`을 실행합니다.

Windows에서는 `FilmFlip.exe`를 실행합니다.

Windows에서 SmartScreen 경고가 나타날 수 있습니다. 이 경우 `추가 정보` → `실행`을 선택하면 사용할 수 있습니다.

------------------------------------------------------------------------

### 2. 폴더 불러오기

다음 두 가지 방법 중 하나를 사용할 수 있습니다.

- `📂 폴더 선택` 버튼 클릭
- 사진이 들어 있는 폴더를 FilmFlip 창으로 드래그 앤 드롭

지원 형식:

- JPG / JPEG
- PNG
- TIFF / TIF
- BMP
- WEBP

폴더를 불러오면 이미지 목록이 먼저 표시되고, 썸네일은 뒤에서 순차적으로 표시됩니다.

------------------------------------------------------------------------

### 3. 사진 순서 변경

사진 순서를 바꾸는 방법은 두 가지입니다.

- 메인 화면에서 행을 드래그하여 원하는 위치로 이동
- `🔄 역순 변경` 버튼으로 전체 순서를 한 번에 뒤집기

필름 스캔본처럼 실제 촬영 순서와 파일 순서가 반대로 들어온 경우에는 `역순 변경` 버튼을 사용하면 됩니다.

------------------------------------------------------------------------

### 4. 사진 확대 보기

썸네일을 더블클릭하면 확대 미리보기 창이 열립니다.

확대 창 조작:

- `←` : 이전 사진
- `→` : 다음 사진
- `ESC` : 닫기

------------------------------------------------------------------------

### 5. 이름 변경

`✏️ 이름 변경` 버튼을 누르면 이름 변경 창이 열립니다.

이름 변경 창에서는 다음 정보를 선택하거나 직접 입력할 수 있습니다.

- 카메라
- 필름
- 현상소
- 장소
- 번호

예시:

```text
FM2_Vision3 250D_망우삼림_001.jpg
FM2_Vision3 250D_망우삼림_002.jpg
FM2_Vision3 250D_망우삼림_003.jpg
```

번호는 자동으로 001부터 생성됩니다.

------------------------------------------------------------------------

### 6. 파일명 구성 변경

이름 변경 창의 `파일명 구성` 영역에서 파일명에 사용할 항목을 선택할 수 있습니다.

- 체크된 항목만 파일명에 사용됩니다.
- `▲ 위로` / `▼ 아래로` 버튼으로 항목 순서를 바꿀 수 있습니다.
- `기본값 복원` 버튼으로 기본 구성을 되돌릴 수 있습니다.

예시:

```text
카메라 + 필름 + 현상소 + 번호
→ FM2_Vision3 250D_망우삼림_001.jpg
```

```text
현상소 + 번호
→ 망우삼림_001.jpg
```

------------------------------------------------------------------------

### 7. 카메라 / 필름 / 현상소 / 장소 목록 관리

각 입력칸 오른쪽의 `⚙` 버튼을 누르면 목록 관리 창이 열립니다.

관리할 수 있는 항목:

- 카메라
- 필름
- 현상소
- 장소

목록 관리 창에서 할 수 있는 작업:

- 항목 추가
- 항목 수정
- 항목 삭제
- `▲ 위로` / `▼ 아래로` 버튼으로 순서 변경

저장된 순서는 드롭다운 목록에 그대로 반영됩니다.

------------------------------------------------------------------------

### 8. 촬영 프리셋 관리

자주 사용하는 촬영 조합은 촬영 프리셋으로 저장할 수 있습니다.

촬영 프리셋에는 다음 정보가 저장됩니다.

- 카메라
- 필름
- 현상소
- 장소
- 파일명 구성
- 정렬 방향

촬영 프리셋에서 할 수 있는 작업:

- 새 프리셋 저장
- 저장된 프리셋 불러오기
- 프리셋 이름 수정
- 프리셋 삭제
- `⚙` 관리창에서 프리셋 순서 변경

프로그램을 종료해도 저장된 프리셋과 순서는 유지됩니다.

------------------------------------------------------------------------

### 9. Undo

이름 변경 후 `↩ Undo` 버튼을 누르면 마지막 이름 변경 작업을 되돌릴 수 있습니다.

주의:

- Undo는 마지막 이름 변경 작업 1회만 되돌립니다.
- 프로그램 종료 후에는 Undo 기록이 유지되지 않을 수 있습니다.
- 파일을 외부에서 이동하거나 삭제한 경우 Undo가 실패할 수 있습니다.

------------------------------------------------------------------------

## v1.1.0 업데이트 내역

이번 업데이트에서는 사용성, 안정성, 성능을 개선했습니다.

### ✨ UI / 사용성 개선

#### 파일명 구성 UI 개선

- 파일명 구성 순서 변경 방식을 드래그에서 `▲/▼` 버튼 방식으로 변경했습니다.
- 드래그 방식보다 안정적으로 순서를 변경할 수 있도록 개선했습니다.
- 안내 문구를 `▲/▼` 버튼 방식에 맞게 수정했습니다.

#### 프로그램 설명 문구 변경

- 메인 화면 상단 설명 문구를 변경했습니다.
- 기존: `필름 스캔 파일 역순 정렬`
- 변경: `필름스캔 파일정리 도구`

#### 촬영 정보 목록 관리 개선

- 카메라 / 필름 / 현상소 / 장소 목록에서 순서 변경 기능을 추가했습니다.
- 각 목록 관리창에서 `▲ 위로` / `▼ 아래로` 버튼으로 항목 순서를 바꿀 수 있습니다.
- 변경된 순서는 드롭다운 목록에 반영되고, 앱을 다시 실행해도 유지됩니다.

#### 촬영 프리셋 관리 개선

- 촬영 프리셋 관리창을 추가했습니다.
- 저장된 촬영 프리셋의 순서를 `▲/▼` 버튼으로 변경할 수 있습니다.
- 촬영 프리셋 관리창에서는 저장된 프리셋 이름만 표시되도록 정리했습니다.

------------------------------------------------------------------------

### ⚡ 성능 개선

#### 폴더 열기 반응 개선

- 폴더를 열 때 이미지 목록을 먼저 표시하고, 썸네일은 뒤에서 순차적으로 표시하도록 개선했습니다.
- 폴더 선택 직후 앱이 멈춘 것처럼 보이는 느낌을 줄였습니다.

#### 썸네일 표시 개선

- 썸네일을 한 번에 모두 생성하지 않고 순차적으로 불러오도록 개선했습니다.
- 썸네일 화질이 과하게 깨지지 않도록 다시 조정했습니다.
- 불필요한 썸네일 로딩 안내 문구를 제거했습니다.

#### 드래그 성능 개선

- 메인 화면에서 행을 드래그할 때 불필요한 화면 갱신을 줄였습니다.
- 드래그 후 행 높이가 줄어드는 문제를 완화했습니다.

#### 이름 변경 창 반응 개선

- 이름 변경 창에서 미리보기 갱신이 불필요하게 여러 번 실행되는 문제를 줄였습니다.
- 촬영 프리셋 적용 시 UI 갱신을 더 안정적으로 처리하도록 개선했습니다.

------------------------------------------------------------------------

### 🐞 버그 수정

- 파일명 구성에서 드래그 시 항목이 사라지는 문제를 수정했습니다.
- 파일명 구성 순서 변경 중 상태가 불안정해질 수 있는 문제를 줄였습니다.
- 메인 화면 썸네일 더블클릭 시 확대창이 중복으로 뜨는 문제를 수정했습니다.
- 썸네일 더블클릭 확대 기능을 꺼도 다시 뜨는 문제를 수정했습니다.
- 드래그 후 썸네일 행 높이가 줄어드는 문제를 수정했습니다.
- 한글 입력 시 마지막 글자가 미리보기에 늦게 반영되거나 누락될 수 있는 문제를 수정했습니다.
- 카메라 / 필름 / 현상소 / 장소 입력값 저장 시 한글 마지막 글자가 잘릴 수 있는 문제를 줄였습니다.
- macOS의 `._` 리소스 파일과 점파일이 목록에 섞이지 않도록 처리했습니다.

------------------------------------------------------------------------

## 주의 사항

- 실제 사진에 적용하기 전에 테스트 폴더에서 먼저 사용해 보는 것을 권장합니다.
- 중요한 사진은 작업 전 백업을 권장합니다.
- 이름이 같은 파일이 이미 존재하는 경우 이름 변경이 실패할 수 있습니다.
- 이름 변경 중에는 프로그램을 강제 종료하지 마세요.
- Undo는 마지막 이름 변경 작업만 되돌릴 수 있습니다.

------------------------------------------------------------------------

## 권장 작업 흐름

1. 원본 사진 폴더를 복사해 테스트용 폴더를 만듭니다.
2. FilmFlip에서 테스트용 폴더를 불러옵니다.
3. 썸네일과 파일 순서를 확인합니다.
4. 필요하면 행 드래그 또는 `역순 변경`으로 순서를 조정합니다.
5. `이름 변경` 창에서 카메라 / 필름 / 현상소 / 장소 정보를 입력합니다.
6. 미리보기를 확인합니다.
7. 문제가 없으면 `OK`를 눌러 이름을 변경합니다.
8. 결과가 이상하면 `Undo`로 되돌립니다.
